# 项目模板路由说明

## 命名规则

- 周报项目模板：`<project-id>-weekly.md`
- 月报项目模板：未来独立维护在月报 skill 中，不放进当前周报 skill

## 当前已支持项目

- `oms-weekly.md`

## 当前项目别名

- `OMS`
- `运营 1 号`
- `运营业务中台`

## 扩展规则

- 新项目必须新增独立 project reference
- 项目专属章节、表头、条线口径、版本口径都写在 project reference 中
- 通用 skill 只负责流程和路由，不维护具体项目事实
